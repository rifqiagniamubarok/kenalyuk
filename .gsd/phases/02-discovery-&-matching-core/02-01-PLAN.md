<task type="auto">
  <name>Add Like, Pass, and Match models to Prisma schema</name>
  <files>prisma/schema.prisma</files>
  <action>
    Add three models to prisma/schema.prisma:
    
    1. Like model:
       - id: UUID primary key
       - userId: String (foreign key to User.id who liked)
       - likedUserId: String (foreign key to User.id who was liked)
       - createdAt: DateTime @default(now())
       - Relations: user (userId), likedUser (likedUserId)
       - Unique constraint: [userId, likedUserId]
       - Indexes: userId, likedUserId, createdAt
    
    2. Pass model:
       - id: UUID primary key
       - userId: String (foreign key to User.id who passed)
       - passedUserId: String (foreign key to User.id who was passed)
       - createdAt: DateTime @default(now())
       - Relations: user (userId), passedUser (passedUserId)
       - Unique constraint: [userId, passedUserId]
       - Indexes: userId, passedUserId
    
    3. Match model:
       - id: UUID primary key
       - user1Id: String (foreign key to User.id)
       - user2Id: String (foreign key to User.id)
       - status: MatchStatus enum (ACTIVE, UNMATCHED)
       - createdAt: DateTime @default(now())
       - updatedAt: DateTime @updatedAt
       - Relations: user1, user2
       - Unique constraint: [user1Id, user2Id]
       - Indexes: user1Id, user2Id, status, updatedAt
    
    Add MatchStatus enum: ACTIVE, UNMATCHED
  </action>
  <verify>npx prisma format succeeds, npx prisma validate passes</verify>
  <done>Like, Pass, Match models defined in schema</done>
</task>
