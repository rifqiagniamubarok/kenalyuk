  <name>Create like API with match creation</name>
  <files>src/app/api/likes/route.ts, src/lib/matching.ts</files>
  <action>
    Create POST /api/likes endpoint:
    - Requires authentication and ACTIVE status
    - Body: { userId, likedUserId }
    - Validate: both users exist and ACTIVE
    - Create Like record with timestamp
    - Check for mutual like: query Like where likedUserId=userId AND userId=likedUserId
    - If mutual: call createMatch(userId, likedUserId)
    - Return: { liked: true, matched: boolean, matchId?: string }
    
    Create src/lib/matching.ts:
    - async createMatch(user1Id, user2Id): creates Match record with status=ACTIVE
    - Returns Match with id and both user IDs
    
    Handle errors: already liked (400), user not found/inactive (404), database errors (500)
  </action>
  <verify>API creates Like, detects mutual likes, creates Match records correctly</verify>
  <done>Like API creates matches when interest is mutual</done>
</task>
