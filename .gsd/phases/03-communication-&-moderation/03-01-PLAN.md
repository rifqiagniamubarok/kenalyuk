  <files>prisma/schema.prisma</files>
  <action>
  Add Message model to schema.prisma with the following fields:
  - id: String @id @default(cuid())
  - matchId: String (foreign key to Match)
  - senderId: String (foreign key to User)
  - content: String @db.Text (message text)
  - isRead: Boolean @default(false) (read status)
  - createdAt: DateTime @default(now())
  - updatedAt: DateTime @updatedAt
  
  Add relations:
  - match: Match @relation(fields: [matchId], references: [id], onDelete: Cascade)
  - sender: User @relation("UserMessages", fields: [senderId], references: [id], onDelete: Cascade)
  
  Add indexes:
  - @@index([matchId])
  - @@index([senderId])
  - @@index([createdAt])
  - @@map("messages")
  
  Add reverse relations to Match and User models:
  - Match: messages Message[]
  - User: messages Message[] @relation("UserMessages")
  </action>
